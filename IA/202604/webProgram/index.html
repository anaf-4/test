주요 기능

파일 업로드

단일 파일 또는 여러 파일을 선택하여 GitHub 저장소에 간편하게 업로드하세요.

폴더 업로드

전체 폴더 구조를 유지한 채로 한 번에 업로드할 수 있습니다.

저장소 선택

업로드할 GitHub 저장소와 브랜치를 자유롭게 선택하세요.

커밋 메시지

각 업로드에 맞는 커스텀 커밋 메시지를 작성할 수 있습니다.

드래그 앤 드롭

파일을 드래그하여 직관적으로 쉽게 업로드하세요.

보안 인증

GitHub OAuth를 통해 안전하게 인증하고 작업하세요.

사용 방법

5단계로 간단하게 시작하세요

1

GitHub 로그인

GitHub 계정으로 안전하게 로그인하세요.

2

저장소 및 브랜치 선택

업로드할 저장소와 브랜치를 선택하세요.

3

파일/폴더 선택

업로드할 파일이나 폴더를 선택하거나 드래그 앤 드롭하세요.

4

커밋 메시지 입력

업로드에 대한 커밋 메시지를 입력하세요.

5

업로드 실행

업로드 버튼을 클릭하여 GitHub에 파일을 업로드하세요.
