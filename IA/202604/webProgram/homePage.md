# GitHub 파일 업로드 홈페이지

## 소개
이 홈페이지는 GitHub 저장소에 파일과 폴더를 간편하게 업로드할 수 있는 웹 기반 도구입니다.

## 주요 기능
- **파일 업로드**: 단일 파일 또는 여러 파일을 선택하여 GitHub 저장소에 업로드
- **폴더 업로드**: 전체 폴더 구조를 유지하며 업로드
- **저장소 선택**: 업로드할 GitHub 저장소 및 브랜치 선택
- **커스텀 커밋 메시지**: 각 업로드에 대한 커밋 메시지 작성
- **드래그 앤 드롭**: 파일을 드래그하여 쉽게 업로드

## 사용 방법
1. GitHub 계정으로 로그인
2. 업로드할 저장소와 브랜치를 선택
3. 파일이나 폴더를 선택 또는 드래그 앤 드롭
4. 커밋 메시지를 입력
5. 업로드 버튼 클릭

## 기술 스택
- **프론트엔드**: HTML, CSS, JavaScript
- **인증**: GitHub OAuth
- **API**: GitHub REST API

## 시작하기
```bash
# 저장소 클론
git clone [저장소 URL]

# 의존성 설치
npm install

# 개발 서버 실행
npm start
```
